onCreate

protected fun onCreate(savedInstanceState: Bundle?): Unit

A method that gets the current firebase user authentication instance. This method also contains a click listener to re-direct the user to the create account page if the user does not have an existing account, a click listener to call the forgot password method, and a click listener to login a user.
