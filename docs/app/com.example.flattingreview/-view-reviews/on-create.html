onCreate

protected fun onCreate(savedInstanceState: Bundle?): Unit

Sets the database reference and collects the path to which the reviews and read from.

On start this method will connect to the database under the 'reviews' path and read all the reviews that are written about a particular flat. It creates a model review for each one that is read from the database and adds them to a list of reviews. When all the reviews are added to the list it calls the createView() method.
